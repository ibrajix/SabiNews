## FEATURES

* MVVM
* Room and DataStore 
* Compose Destinations - https://github.com/raamcosta/compose-destinations
* Hilt
* Pagination 3
* Retrofit
* Landscapist - https://github.com/skydoves/Landscapist
* Offline Capabilities - Using Remote Mediator
* Light and Dark Mode
* Search
